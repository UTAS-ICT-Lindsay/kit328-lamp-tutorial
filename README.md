# KIT328 Tutorial 1A -- First PHP App
This codebase contains a simple PHP app in the `frontend` folder. You can find the code in 

The code is served by Apache in a docker container, and when run through GitHub Codespaces, will automatically be launched.

It has a simple [frontend/index.php](frontend/index.php) file.

## Running the Code
When run through GitHub Codespaces, will automatically be launched. Click the **View in Browser** button in the prompt or use the *Ports* tab to see the running application.

However if you wish to run the docker container manually, you can do so by running this command:
```sh
docker compose up -d
```
